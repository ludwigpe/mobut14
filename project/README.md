# Web-server

Dependencies
* node v0.10.13 or higher
* npm 1.3.2 or higher

## Setup
``` npm install ```
## Run
``` npm start ```
## Develpoment
```
  DEBUG="web-server" node-dev index.js
```
This will restart the server when changes are made in the underlaying files. This will require you to first run ``` npm install -g node-dev ```
